In my 6th semester, I worked on a project where I created a webpage to calculate the busfare of bus stops along the ringroad of Kathmandu Valley. The project required users to first sign up and then login to access the platform's features. Once logged in, users could select their pickup and destination points from convenient dropdown menus, fill in necessary details and calculate the total distance between the source and destination to determine the fare. The total distance and the fare required, both are displayed.

The project was crafted using HTML and CSS for designing and styling. The Project also utilized JavaScript for implementing dynamic functionalities such as form validation, distance calculation and user interaction. For the backend database, I used MongoDB to store the user information securely (Email and Passwords) for signup and login functionalities. Through this a user can sign up and login only with the information already stored into the database.
This project helped me learn more about Web Development and database.
